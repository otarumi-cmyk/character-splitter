# キャラクター自動分割ツール インストールガイド

このツールをローカルでインストールして使う方法です。

## 必要な環境

- **Python 3.8以上** がインストールされていること
- **インターネット接続**（初回インストール時に依存関係をダウンロードします）

## インストール方法

### macOS / Linux の場合

1. **ターミナルを開く**

2. **このフォルダに移動**
   ```bash
   cd ~/character-splitter
   ```

3. **インストールスクリプトを実行**
   ```bash
   ./install.sh
   ```
   
   初回は数分かかります（依存関係のダウンロードのため）

4. **起動**
   ```bash
   ./start.sh
   ```

5. **ブラウザで開く**
   - `http://localhost:8000` を開く

### Windows の場合

1. **このフォルダを開く**

2. **`install.bat` をダブルクリック**
   
   コマンドプロンプトが開いて、自動でインストールが始まります。
   初回は数分かかります。

3. **`start.bat` をダブルクリック**
   
   サーバーが起動します。

4. **ブラウザで開く**
   - `http://localhost:8000` を開く

## 使い方

1. ブラウザで `http://localhost:8000` を開く
2. 画像をアップロード
3. 「キャラクターを自動分割」をクリック
4. 結果をダウンロード

## トラブルシューティング

### 「Python が見つかりません」エラー

- Python 3.8以上をインストールしてください
- https://www.python.org/downloads/

### 「pip install が失敗する」エラー

- インターネット接続を確認
- ファイアウォールがブロックしていないか確認
- 管理者権限で実行してみる

### 「ポート8000が既に使われている」エラー

- 他のアプリケーションがポート8000を使っている可能性があります
- `start.sh` または `start.bat` を編集して、別のポート（例: 8001）に変更してください

### メモリ不足エラー

- 大きな画像を処理する際にメモリ不足になる場合があります
- 画像サイズを小さくするか、メモリを増やす

## アンインストール

このフォルダを削除するだけでOKです。

```bash
rm -rf ~/character-splitter  # macOS/Linux
```

Windows の場合は、フォルダを削除してください。
