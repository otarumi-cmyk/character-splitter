### 使用エンドポイント
- `POST /save-result` - セッションIDで結果保存
- `GET /get-result` - セッションIDで結果取得
- `POST /save-result-by-lineuser` - LINE User IDで結果保存
- `GET /get-result-by-lineuser` - LINE User IDで結果取得
- `POST /claim-pending-result` - PENDING結果をLINEユーザーにリンク
- `GET /debug` - デバッグ情報

## 配布タグ条件

以下の**すべて**を満たす場合に配布対象：

1. **年齢**: 20〜34歳
2. **住まい**: 東京都、神奈川県、埼玉県、千葉県、大阪府、福岡県、愛知県
3. **転職時期**: 1年以内（1ヶ月以内、3ヶ月以内、6ヶ月以内、1年以内）
4. **転職回数**: 2回以下（1社目、2社目、3社目）

## ダッシュボード機能

WordPress管理画面 → 転職可能性診断

- **サマリー**: 診断総数、配布タグ対象数、回答分布
- **ユーザー一覧**: LINE ID、LINE名、年齢、住まい、転職時期、転職回数、配布タグ
- **CSVエクスポート**: 全データまたは配布対象のみ
