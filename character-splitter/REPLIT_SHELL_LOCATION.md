# Replitの「Shell」タブの場所

## Shellタブの見つけ方

### パターン1: 画面下部にある場合（最も一般的）

Replitの画面を開いたら：

1. **画面の下の方**を見てください
2. 以下のようなタブが並んでいます：
   ```
   [Console] [Shell] [Problems] [Debugger]
   ```
   または
   ```
   [Shell] [Console] [Terminal]
   ```
3. **「Shell」**というタブをクリック

### パターン2: 右側のパネルにある場合

1. 画面の**右側**を見てください
2. 縦に並んだタブの中に「Shell」がある場合があります
3. クリックして開く

### パターン3: メニューから開く場合

1. 画面上部のメニューバーを見る
2. 「Tools」や「View」メニューをクリック
3. 「Shell」や「Terminal」を選択

## 見つからない場合の対処法

### 方法A: キーボードショートカット

1. `Ctrl + J`（Windows/Linux）または `Cmd + J`（Mac）を押す
2. これでShell/ターミナルが開きます

### 方法B: 直接コマンドを実行

1. 画面中央のコードエディタの下に「Console」というタブがある場合
2. そこに直接コマンドを入力できる場合があります
3. または、画面下部の入力欄に直接入力

### 方法C: 画面を探す

Replitの画面構成：
```
┌─────────────────────────────────────┐
│  [Run] [Stop] ... [Share]          │ ← 上部のツールバー
├─────────────────────────────────────┤
│                                     │
│  コードエディタ（main.pyなど）      │ ← 中央
│                                     │
├─────────────────────────────────────┤
│ [Console] [Shell] [Problems] ...   │ ← 下部のタブ ← ここ！
│ $                                   │ ← コマンド入力欄
└─────────────────────────────────────┘
```

## Shellタブを開いたら

1. Shellタブをクリック
2. 画面下部に以下のような表示が出ます：
   ```
   $ 
   ```
   または
   ```
   > 
   ```
3. そこに以下を入力してEnterキーを押す：
   ```bash
   pip install -r requirements.txt
   ```

## それでも見つからない場合

**代替方法：Runボタンで自動インストール**

1. Replitは自動的に `requirements.txt` を検出してインストールを試みます
2. まず「Run」ボタンをクリックしてみてください
3. エラーが出た場合、そのエラーメッセージを見て対処

## 確認方法

Shellタブが開いているかどうか：
- ✅ 画面下部に `$` や `>` という記号が見える
- ✅ コマンドを入力できるカーソルがある
- ✅ 「Shell」「Terminal」「Console」という文字が見える
