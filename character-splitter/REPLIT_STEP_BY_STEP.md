# Replit デプロイ手順（画面付き）

## 方法1: 直接URLでインポート（最も簡単）

1. https://replit.com にアクセスしてログイン
2. 画面左上の「+ Create Repl」ボタンをクリック
3. 開いた画面で、**上部の検索バーまたはURL入力欄**に以下を入力：
   ```
   github.com/otarumi-cmyk/character-splitter
   ```
4. Enterキーを押すか、「Import」ボタンをクリック

## 方法2: メニューから選択

1. https://replit.com にアクセスしてログイン
2. 画面左上の「+ Create Repl」ボタンをクリック
3. 開いたモーダル/画面で：
   - **左側のメニュー**に「Import from GitHub」というタブ/ボタンがある場合
   - または、**上部のタブ**に「Import」がある場合
   - または、**「From GitHub」**というボタンがある場合
4. 見つけたらクリックして、リポジトリURLを入力：
   ```
   https://github.com/otarumi-cmyk/character-splitter
   ```

## 方法3: 直接URLでアクセス

以下のURLに直接アクセス（ログイン後）：
```
https://replit.com/github/otarumi-cmyk/character-splitter
```

## 見つからない場合の対処法

### パターンA: 「Create Repl」をクリックした後の画面
- 画面中央に「Search for a template」と表示されている場合
- その検索バーに `github.com/otarumi-cmyk/character-splitter` と入力
- または、画面下部に「Import from GitHub」リンクがある場合

### パターンB: 左側にサイドバーがある場合
- 「Templates」「My Repls」などのタブがある
- その中に「Import」や「GitHub」というタブを探す

### パターンC: 画面が変わった場合
1. 画面右上の「Settings」（歯車アイコン）をクリック
2. 「Import」や「GitHub」の項目を探す

## スクリーンショットで確認したい場合

Replitの画面で以下を探してください：
- ✅ 「Import」という文字
- ✅ 「GitHub」という文字
- ✅ 「From GitHub」というボタン
- ✅ URL入力欄（検索バーのようなもの）

## それでも見つからない場合

**方法3の直接URLアクセスが最も確実です：**
1. https://replit.com にログイン
2. ブラウザのアドレスバーに以下を入力：
   ```
   https://replit.com/github/otarumi-cmyk/character-splitter
   ```
3. Enterキーを押す

これで自動的にインポートが開始されます。
